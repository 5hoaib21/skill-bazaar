## Backend API Endpoints

### Opportunities
- `GET /api/opportunities` ŌĆö List with search, filter, sort, pagination
- `GET /api/opportunities/:id` ŌĆö Detail
- `POST /api/opportunities` ŌĆö Create (auth required)
- `PATCH /api/opportunities/:id` ŌĆö Update (owner only)
- `DELETE /api/opportunities/:id` ŌĆö Delete (owner only)
- `GET /api/opportunities/my` ŌĆö Current user's opportunities

### Applications
- `POST /api/applications` ŌĆö Submit application
- `GET /api/applications/my` ŌĆö My applications (volunteer)
- `GET /api/applications/received` ŌĆö Received applications (organizer)
- `PATCH /api/applications/:id/approve` ŌĆö Approve (organizer)
- `PATCH /api/applications/:id/reject` ŌĆö Reject (organizer)
- `PATCH /api/applications/:id/withdraw` ŌĆö Withdraw (volunteer)
- `PATCH /api/applications/:id/complete` ŌĆö Mark complete (organizer)

### Categories
- `GET /api/categories` ŌĆö List active categories

### Contact
- `POST /api/contact` ŌĆö Submit contact form (rate limited)

### Dashboard
- `GET /api/dashboard/summary` ŌĆö Aggregated stats for charts

### Auth
- `GET /api/auth/me` ŌĆö Current user info
